# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звёздочками (*). 
Например, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звёздочками (**). 
Например, **вот так**
