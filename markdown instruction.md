## Списки

Чтобы доабвить ненумерованные списки, необходимо пункты выделить (*). Например, вот так:
* эелемент 1
* элемент 2
* элемент 3

Чтобы добавить нумерованные списки необходимо пукнты просто пронумеровать. Например вот так:

1. Первый пункт
2. Второй пункт
